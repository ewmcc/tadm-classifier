# tadm-classifier

Exploratory analysis of Hamilton TADM pressure curves and gravimetric weights, with the long-term goal of training a classifier that predicts pipetting quality directly from waveform shape.

**[tadm-classifier.ipynb](tadm-classifier.ipynb)** - a Jupyter notebook covering data loading, curve decoding, weight-to-volume conversion, labelling, and EDA plots (volume distribution and raw pressure waveforms).
